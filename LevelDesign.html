The Project

This was my first ever level design project, me and two of my classmates got to create a level together working almost from scratch. We got to pick one of three level types to create and we chose to make a 3D puzzle Platform tutorial level. We did so because of the challenge of making a tutorial level for a platformer but also the creative freedom it provided. The project provided us with assets we could use and a starter world with gameplay elements as a movable character and some puzzle mechanics, but we also created our own mechanics using Unreal Engines visual scripting, Blueprints.

In this level we all worked together, we built this level in unreal engine and therefor used their multiuser editing to work in the same level at the same time. So we all had an say in the level design. We did though work on different things in the level. I for example created the starter house you begin the level in, I used the assets that were provided to create a fully decorated house, both the outside and inside!
I personally also worked alot with blueprints during this project, I created all the blueprints for our puzzles, moving platforms and doors. I also managed to create a system that calculates the players distance to the portal in the level, and changed the skys color depending on it. We were supposed to make the world seem more magicall the closer you got to the portal, So i made this system where the sky becomes more purple the closer the player gets to it.
